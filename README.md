# PS教程
Adobe Photoshop 教程 —— 一部来自于元卅馆的《元卅PS教程》。

本教程以实用为主，每章节都让你学会一个新的技能，并将原理用通俗易懂的方式对你娓娓道来。

在Bilibili和YouTube发布有视频教程。在官网[元卅馆](www.yuansasi.com)有对应文字讲义。
